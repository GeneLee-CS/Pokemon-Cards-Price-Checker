# Pokémon Card Price Tracker
An end-to-end **data engineering** project that collects, processes, stores, and visualizes historical Pokémon card sale prices from eBay.  

---

## Tech Stack
- Python
- Poetry
- GitHub CI/CD (later)
- AWS S3 (later)
- dbt & Airflow (later)
